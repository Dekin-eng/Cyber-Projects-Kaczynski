> **Keep your passwords safe, organized, and accessible - all with one master password**

PassIt is a lightweight, local-only password manager that stores your credentials securely using AES-256 encryption. No cloud storage, no telemetry, just your passwords under your control.

## ✨ Features

- 🔒 **AES-256 Encryption** - Military-grade encryption for your vault
- 🔑 **One Master Password** - Remember one password, access all others
- 📋 **Auto-Clear Clipboard** - Copied passwords disappear after 10 seconds
- 💻 **Local-Only Storage** - Your data never leaves your computer
- 🎲 **Password Generator** - Create strong, random passwords
- 🔍 **Quick Search** - Find any login in seconds
- 📁 **Categories & Tags** - Organize passwords by project or type
- 💾 **Auto-Save** - Never lose changes

## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/Dekin-eng/passit.git

# Go to the project folder
cd passit

# Install required packages
pip install -r requirements.txt

# Run PassIt
python passit.py
